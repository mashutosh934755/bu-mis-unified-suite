# BU MIS Unified Suite

**BU MIS Unified Suite** is a WordPress-based Library Management Information System dashboard designed for academic libraries. It integrates operational library data, repository analytics, journal publishing metrics, room booking statistics, plagiarism workflow indicators, and AI-assisted dashboard interpretation into a single modular dashboard.

> Developed as a case-study architecture for Bennett University Learning Resource Centre.

## Key Features

- WordPress admin dashboard for library staff
- Public read-only dashboard interface
- Koha / library-at-a-glance statistics
- Daily circulation overview
- PTC / footfall analytics
- DSpace repository analytics
- OJS publication analytics
- GD / meeting room booking analytics
- Plagiarism checking workflow KPIs
- Optional AI assistant for dashboard interpretation
- Modular structure for future extension
- Configurable API endpoints and credentials
- GitHub-ready documentation and security guidelines

## Dashboard Sections

1. Library at a Glance  
2. Today Overview  
3. PTC Dashboard  
4. DSpace Dashboard  
5. OJS Dashboard  
6. GD Room Dashboard  
7. Plagiarism Checking Dashboard  
8. Optional AI Assistant  

## Repository Status

This repository is prepared as a **clean GitHub starter package**. Add your actual plugin code into the appropriate folders:

```text
includes/
templates/
assets/
bu-mis-unified-suite.php
```

Before publishing, remove all private data such as internal IP addresses, API keys, passwords, tokens, and institution-only endpoints.
