# Modules

## Koha / Library at a Glance
Shows active patrons, total items, total titles, checked-out books, and e-resources.

## Today Overview
Tracks fine paid, no dues, issues, returns, and new members.

## PTC Dashboard
Tracks library entry / footfall data.

## DSpace Dashboard
Tracks repository communities and scoped thesis counts.

## OJS Dashboard
Tracks journals and published articles.

## GD Room Dashboard
Tracks room-wise booking count.

## Plagiarism Dashboard
Tracks pending, completed, total, and staff-wise report processing.

## AI Assistant
Optional dashboard interpretation layer.
